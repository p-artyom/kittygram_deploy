# Kittygram

## Описание

Kittygram — социальная сеть для обмена фотографиями любимых питомцев. Это
полностью рабочий проект, который состоит из бэкенд-приложения на Django и
фронтенд-приложения на React.

## Технологии

- Python 3.10.6
- Django 3.2.3
- Django REST framework 3.12.4
- Gunicorn 20.1.0
- Nginx 1.18.0
- Node.js 18.16.1
